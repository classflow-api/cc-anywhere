## 边界值核验

| 边界 | Mac 端处理 | Phone 端处理 | 一致? |
|------|-----------|-------------|------|
| filename = `IMG_001.jpg` | `<uid>.jpg` | `<uid>.jpg` | ✅ |
| filename = `archive.tar.gz` | `<uid>.gz`(NSString.pathExtension 只取最后段) | `<uid>.gz`(lastIndexOf 同样取最后段) | ✅ |
| filename = `noext` | `<uid>`(ext empty) | `<uid>`(没 `.` ext 空) | ✅ |
| filename = `file.`(末尾 `.`) | `<uid>`(NSString 返回空) | `<uid>`(substring(len) 空字符串) | ✅ |
| filename = `.bashrc`(dotfile) | `<uid>`(NSString 把开头 `.` 视为隐藏,无 ext) | `<uid>.bashrc`(lastIndexOf('.')=0, substring(1)='bashrc') | ❌ **不一致** |
| filename = `测试.jpg`(中文) | `<uid>.jpg` | `<uid>.jpg` | ✅ |
| filename = `My Photo.jpg`(空格) | `<uid>.jpg` | `<uid>.jpg` | ✅(空格被 upload_id 替代) |
| filename = `name."double".png`(引号) | `<uid>.png` | `<uid>.png` | ✅ |

**唯一发现的不一致:dotfile**。详见警告级新问题 #3。
