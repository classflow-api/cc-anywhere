# 行为准则 · Code of Conduct

本项目采用 [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) 作为社区行为准则。

## 我们的承诺

身为社区成员、贡献者、维护者，我们承诺让所有人参与本项目的体验不受骚扰，无论年龄、体型、可见或不可见的残障、族裔、性别认同、经验水平、教育、社会经济地位、国籍、外貌、种族、宗教、性取向。

我们承诺以促进开放、友善、多元、包容、健康社区的方式行事和互动。

## 我们的标准

**有助于建立积极环境的行为**：

- 对他人保持同理心和善意
- 尊重不同意见、观点和经验
- 给出和优雅接受建设性反馈
- 对自己的错误承担责任、向受影响者道歉、从经验中学习
- 关注对整个社区最有益的事情，而不仅仅是个人

**不可接受的行为**：

- 使用性化的语言或图像，以及任何形式的性关注或挑逗
- 挑衅、侮辱或贬低性评论，以及人身或政治攻击
- 公开或私下的骚扰
- 未经明确许可发布他人的私密信息（实际地址、电子邮件等）
- 在专业环境中其他可被合理视为不当的行为

## 执行责任

社区维护者负责澄清和执行可接受行为的标准，并对他们认为不当、威胁、冒犯或有害的行为采取适当公平的纠正措施。

社区维护者有权移除、编辑或拒绝评论、提交、代码、wiki 编辑、issue 和其他贡献，对于违反此行为准则的内容，并将适时告知调节决定。

## 适用范围

此行为准则适用于所有项目空间，也适用于个人代表项目或社区出现在公共场所的情况。

## 执行

辱骂、骚扰或其他不可接受的行为可向项目维护者报告，邮箱：**opensource@yoolines.com**。所有投诉将得到审查和调查，并产生认为必要和适当的回应。

社区维护者有义务对事件报告人的身份保密。

## 致谢

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org)，[版本 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct.html)。

## English

This project adopts the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/) as its community Code of Conduct.

In short:

- Be kind, empathetic, and respectful
- Welcome differing viewpoints and experiences
- Give and gracefully accept constructive feedback
- Take responsibility for mistakes
- Focus on what's best for the community

Unacceptable: harassment, derogatory comments, personal/political attacks, sharing private info without consent, sexualized language or imagery.

Report violations to **opensource@yoolines.com**. All reports are confidential.
