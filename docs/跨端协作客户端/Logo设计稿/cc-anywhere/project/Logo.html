Concept · A

Sync Loop

两个 c 互为镜像, 中间一个跳动的光标块 —— Mac 与手机之间的实时同步. 最直接地表达 "cc" 与 "双端协作".

Bracket Terminal

一对方括号包住一个光标块 —— 把整个终端封装成一枚徽章. 最 "工程师", 最贴近 shell prompt 的视觉.

Relay Chevrons

两个 chevron 套娃 · 大的是 Mac, 小的是手机, 信号从大端转发到小端 — 最直白地表达"接力"和"远程接管".
